# HTML

## Definition
HTML is the standard markup language for Web pages

HTML is **NOT** a programming language

## HTML Elements
### Header
```
<h1> header1</h1>
<h2> header2</h2>
```
### Paragraph
`<p>Text here</p>`

Change text fonts:
```
<i>Italic</i>
<u>Underline</u>
<b>Bold</b>
```

### Section
```
<section>
    elements...
</section>
```

### Lists
```
Unordered list
<ul>
    <li> Apple </li>
    <li> Banana </li>
</ul>

Ordered list
<ol>
    <li> Apple </li>
    <li> Banana </li>
</ol>
```

### Links
```
<a href="index.html"> Click here </a>
```

### Images
```
<img src="image.png" width="100" height="50" alt="Image Name">
```

### Table
```
<p>
<table>
    <tr>
        <th> head1 </th>
        <th> head2 </th>
    </tr>
    <tr>
        <td> item1 </td>
        <td> item2 </td>
    </tr>
</table>
</p>
```

### Form
```
<form>
    <lable> Type in Name</lable>
    <input name="Name"><br>
    <lable> Date</lable>
    <input name="Date" type="date><br>
    <button>Click</button>
</form>
```
### Commonly used
Insert a new line:
```
<br> New line starts here
```

Comments:
```
<!-- Comments -->
```
